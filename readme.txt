# Stranger Questions - FAQ

*Stranger Questions - FAQ is a free WordPress plugin. The plugin is very user-friendly and easy to use. It will help you add a frequently asked questions block.
You can also add an image to the answer to the question. And you can use the basket for questions. A block of questions can be taken out in a convenient place with the help of a shortcode. I hope this plugin is useful to you and use this plugin.*

## Plugin features

- Add a block of frequently asked questions.
- Adding an image to an answer.
- Output using shortcode.
- Delete questions in the basket.


== Installation ==

Install the plugin

1. Upload the entire "Stranger Questions - FAQ" folder to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Open the "FAQs" section.
1. Click "Add New" and fill in the appropriate fields. Then click publish.
1. To output, a block of questions in the right place, enter the following shortcode: [sq-faq].
